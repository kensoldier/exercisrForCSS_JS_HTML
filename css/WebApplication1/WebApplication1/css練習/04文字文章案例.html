今天天氣真的好好要不要出外走走

2017年08月22日18:54 旅遊新聞 收藏

金門、馬祖每年接待許多透過小三通來旅遊的大陸旅客，是少數不受兩岸關係影響觀光的縣市。旅行業界近期盼推動 「跳島郵輪」，讓陸客搭郵輪到金馬後再到澎湖旅遊，帶動澎湖觀光人潮。交通部航港局正在盤點港口條件與資源， 透露廈門、泉州業者也有意願合作。

兩岸關係不佳，陸客從2015年的400萬人次高峰，重挫至去年的270萬人次，但透過小三通到金門旅遊的陸客人次不減反增， 去年正式突破70萬大關，馬祖則每年穩定維持5至7萬人次。澎湖雖在小三通開放範圍，但每年百萬旅遊人次陸客數不到5萬， 主要是澎湖離中國大陸較遠，少有陸客特地到澎湖旅遊。
